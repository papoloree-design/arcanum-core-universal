# AION Edge Worker

Worker para edge computing compatible con Cloudflare Workers.

## Desarrollo Local

```bash
node server-stub.js
```

## Deployment a Cloudflare

1. Configurar wrangler.toml con tu account_id
2. Autenticar: `wrangler login`
3. Deploy: `wrangler deploy`

## Características

- Latencia ultra-baja
- Caché distribuido
- Rate limiting
- Request routing
